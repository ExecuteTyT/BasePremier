### 1.3. ТЗ на фотосессию мастеров (C-01)

> **Это самый дорогой блокер. Если фотосессии не будет — страница `/barbers` теряет 70% эмоциональной силы. Единый стиль критичен.**

**Бюджет.** В Казани профессиональный портретист с обработкой берёт 25–50 тыс ₽ за смену 4–6 часов (10 портретов укладываются с запасом). Альтернатива бюджетнее (15–25 тыс ₽) — выпускник школы фотографии. Awwwards-уровень — только первый вариант.

**Где снимать.** На Шаляпина 26, в зале или у фактурной стены интерьера. **Не** в студии с белым циклом (чужой контекст).

**Стиль.**
- Чёрный фон (тёмная стена интерьера или подбранный тканевый фон) — портреты будут на тёмном фоне сайта.
- Низкий ключ освещения (key light с одной стороны, fill минимальный, accent на волосы).
- Глубина резкости умеренная (не open f/1.4) — хочется, чтобы и лицо, и руки в фокусе для часто используемых половинных портретов.
- Цветовая температура 4500–5000K. **Не** тёплая. **Не** холодная синяя.
- Без эффектов «бьюти-ретуши» в пост-обработке — характерные черты, мимика, морщины, борода — нужны.

**Кадры на каждого мастера** (минимум):
1. Фронтальный портрет, плечи в кадре — для главной карточки.
2. Полупрофиль с инструментом в руках — для hover/секондари кадра.
3. Силуэт в работе с клиентом (бэкграунд блюрный, в фокусе руки) — для секции «Процесс».

**Технические требования** для веб:
- Исходник минимум 4000×6000 px, формат RAW/TIFF + JPEG.
- Цветовое пространство sRGB после обработки.
- Имена файлов: `barber-{slug}-portrait.jpg`, `barber-{slug}-action.jpg`, `barber-{slug}-process.jpg`.
  - Slugs: `marat`, `vyacheslav`, `sayod`, `alexey`, `timerlan`, `nikolay`, `dzhim`, `arina`, `murat`, `diana`.
- Заказчик передаёт исходники → Claude Code обрабатывает в `next/image` пайплайне (AVIF + WebP, размеры 480w/768w/1280w/1920w).

**Согласия.** Нужно письменное согласие каждого мастера на:
- Использование портрета на сайте `basepremier.ru`
- Использование в Я.Директ-баннерах
- Использование в постах в соцсетях бренда
Шаблон согласия — Claude Code сгенерирует по запросу `/gen consent-barber`.

### 1.4. ТЗ на использование существующего контента

**Интерьер (5–10 общих + 5–10 деталей).** Используем:
- 1 кадр в hero (если решим не использовать видео — план B)
- 4–6 кадров в галерею интерьера на главной (секция `interior-gallery`)
- 2–3 кадра как декоративные фоны в footer и `/about`
- Разрешение исходников должно быть ≥ 3000 px по длинной стороне. Если меньше — пересмотреть и попросить переслать оригиналы.

**Косметика (5–10 кадров).**
- 4 кадра в секцию «Процесс / стандарты» — рядом с упоминанием брендов Graham Hill, Davines, London Grooming Co, Solomon's.
- 1 кадр в детальную карточку услуги «Премиальный уход London Grooming».
- 1 кадр в карточку «Премиальное моделирование SOLOMON'S».
- 1 кадр в карточку «Детокс кожи головы Graham Hill».

**Процесс стрижки (5–10 кадров).**
- 3–4 кадра в секцию «Процесс» на главной (3 шага: подготовка → стрижка → финал).
- 2–3 кадра в `/about` как иллюстрация ритуала визита.

**Все фото проходят** через `next/image` с `placeholder="blur"` (генерируем blur-data-url через `plaiceholder`) и `loading="lazy"` для всего, что ниже первого экрана. На первом экране — `priority`.

## 3. MANIFESTO БРЕНДА

> Идёт на странице `/about` единым потоком. Не разбит на буллеты. Это «эссе» на 350–400 слов.

```markdown
# О нас

BASE Premier — премиальный барбершоп в Казани.
Мы открылись в 2022 году в самом центре города,
на улице Шаляпина, 26 — в трёхстах метрах от
Концертного зала Филармонии.

Мы делаем три вещи:
стрижём, ухаживаем, заботимся.

И ничего больше.

В этом — суть. У нас нет караоке, бильярда,
кальянной комнаты, дорогого виски и притворной
мужской атмосферы. Если вам нужно мужское
пространство в стиле американского клуба — мы
посоветуем коллег. Здесь — другое.

Здесь тишина. Свет, который не режет глаз.
Музыка, которую можно не слушать. Один час,
в течение которого десять человек команды
работают для одного клиента.

В нашем зале — десять мастеров, каждый из которых
за плечами имеет от ста до трёхсот стрижек.
В работе мы используем профессиональные средства
от Graham Hill, Davines, The London Grooming Co и
Solomon's — те же, которыми пользуются в Лондоне,
Милане и Берлине.

Каждая стрижка длится не меньше часа. В этот час
входит детокс кожи головы, мытьё, стрижка по
структуре волос, моделирование, укладка, бритьё
шеи. Если у вас борода — добавляется получасовая
работа над контурами и компресс.

Мы знаем, что у людей, которые приходят к нам, —
бизнес, переговоры, дети, тренажёры, утренние
кофе и вечерние самолёты. Поэтому мы держим зал
открытым семь дней в неделю, с десяти утра до
девяти вечера. Записаться можно за один клик —
прямо здесь, на сайте.

Мы не обещаем «преобразить вас». Мы обещаем час
покоя и чёткую, продуманную работу. Это всё,
что нужно. Этого, как правило, достаточно.

— BASE Premier
   ул. Шаляпина, 26 · Казань · с 2022
```

**Длина.** ~370 слов на русском, ~2200 знаков. Помещается на одну страницу `/about` без скролла на десктопе при `font-size: 1.25rem`, `line-height: 1.6`, max-width 640 px.

**Стиль.**
- Местами рваный ритм (короткие предложения после длинных).
- Перечисления через точки в строку, не через буллеты — это часть фирменного стиля.
- **Не редактировать без согласования.** Если заказчик хочет изменить — Claude Code предлагает альтернативу с пометкой `[v2]` в комментарии и сохраняет историю.

## 6. ОБОСНОВАНИЕ ЦЕНЫ — 4 ТЕЗИСА

> Идёт на главной странице в секции «Стандарты» (или на странице `/services` как закреплённый блок). Объясняет, почему стрижка стоит 2 700 ₽, а не 800 ₽ как в массмаркете.

### Тезис 1. Час, а не 30 минут

```
Большинство барбершопов выделяют 30 минут на стрижку. Мы — час.
В этот час входит детокс кожи головы, мытьё, стрижка по структуре
волос, моделирование, укладка и завершающее парфюмирование.
В среднем — в два раза больше работы.
```

### Тезис 2. Профессиональная косметика, а не один шампунь

```
В нашем зале на полках — Graham Hill, Davines, The London Grooming Co
и Solomon's. Шампунь для тонких волос, шампунь для жирной кожи головы,
тоник с никотиновой кислотой, восемь видов укладочных средств.
Под вашу структуру волос мы подбираем средство, а не используем
универсальное.
```

### Тезис 3. Десять мастеров, не один универсал

```
В нашем зале десять мастеров. Каждый из них — со своим почерком и
любимыми техниками. Если ваша задача — fade с короткими висками и
длинной макушкой, к Сайоду вы попадёте быстрее, чем к универсальному
мастеру в массмаркете. Мы не назначаем мастера автоматически —
вы выбираете.
```

### Тезис 4. Тишина, а не open space

```
В нашем зале — пять рабочих мест и десять минут между визитами.
Это значит: вы не сидите рядом с другим клиентом «локоть в локоть»,
вас не отвлекает разговор соседа, и мастер не торопится — он работает
с вашими волосами, а не «успевает до следующего».
```

### Применение в дизайне

В DESIGN-BRIEF.md эти 4 тезиса — четыре карточки в горизонтальном scroll или анимация cross-fade. Каждая карточка: цифра 01/02/03/04, заголовок, текст. **Без иконок** (банально и не в стиле Tom Ford).

## 11. ОТЗЫВЫ

### 11.1. Источник

- **Я.Карты:** 394 отзыва, рейтинг 5.0. Берём 5–7 лучших по объёму и эмоциональной силе.
- **2ГИС:** дополнительно 30–50 отзывов, тоже хорошие.
- **Instagram:** комментарии под постами (только если заказчик подтвердит).

### 11.2. Что показываем

> Заказчик подтвердил «только цитаты с Яндекс.Карт без имён». Поэтому **анонимизируем** во всех случаях.

```yaml
review:
  - text: |
      [цитата клиента, 80–200 знаков]
    sourceLabel: Яндекс.Карты
    sourceDate: 2025-09-12      # для микро-данных Schema.org
    rating: 5
    serviceContext: муж. стрижка   # опционально, если упоминается
    isAnonymous: true
```

### 11.3. Где показываем

- **Главная:** секция «Отзывы» — горизонтальный scroll или 3-в-ряд карточек, 5 шт.
- **Страница услуги:** 2 отзыва, контекстно отфильтрованных по `serviceContext`.
- **Страница мастера:** 2–3 отзыва, упоминающих имя мастера.
- **JSON-LD:** все отзывы вшиваем в `AggregateRating` для главной (`aggregateRating: 5.0, reviewCount: 394`).

### 11.4. Процесс выбора

> Это блокер C-03 в § 1.2.

1. Айрат выгружает 30 лучших отзывов с Я.Карт (ручным копированием или через API Я.Карт, если есть доступ).
2. Claude Code предлагает 7 финалистов — самые эмоционально сильные, без имён, длиной 80–200 знаков.
3. Айрат подтверждает.
4. Финалисты идут в Sanity → на сайт.

- Ногтевой зал: Массаж рук (500₽) → ... → Педикюр полный (3000₽).

### 12.3. Поиск по прайсу

На странице `/services` — поле поиска по `title` и `shortDescription`. Реализация — `fuse.js` (~5 KB gzip), без бэка.

## 13. ТИПОГРАФИКА НА СТОРОНЕ РАНТАЙМА

### 13.1. Библиотека

Используем `typograf` (npm package, RU-русский). Каждый текст в Sanity и каждый MDX-файл проходит через типограф **во время билда**, не в рантайме (для производительности).

```ts
// src/lib/typograf.ts
import Typograf from 'typograf';

const tp = new Typograf({ locale: ['ru', 'en-US'] });
tp.enableRule('common/space/nbsp-before-particle');
tp.enableRule('ru/dash/main');
tp.enableRule('ru/punctuation/quote');
tp.enableRule('ru/nbsp/afterShortWord');

export function typograf(text: string): string {
  return tp.execute(text);
}
```

### 13.2. Что делает типограф автоматически

- Заменяет `"кавычки"` на «ёлочки».
- Внутренние кавычки — „«лапки»“.
- Длинное тире вместо `--`.
- Неразрывные пробелы между числом и единицей: `1 800 ₽`, `1 ч`, `30 мин`.
- Неразрывные пробелы после коротких слов («в», «на», «по», «к», «о»).
- Неразрывные пробелы перед предлогами и союзами.
- Замена `(c)` на ©, `(r)` на ®.

### 13.3. Что НЕ делает (оставляем как есть)

- Не меняет регистр.
- Не исправляет грамматику.
- Не сокращает текст.
- Не фильтрует «стоп-слова».

### 13.4. Применение

```ts
// При импорте Sanity-документа
const article = await client.fetch(query);
article.body = typograf(article.body);
article.title = typograf(article.title);
```

```mdx
{/* MDX-файлы блога */}
import { typograf } from '@/lib/typograf';
import meta from './meta.json';

export const title = typograf(meta.title);
```

### 13.5. Тест

```ts
// src/lib/typograf.test.ts
import { describe, it, expect } from 'vitest';
import { typograf } from './typograf';

describe('typograf', () => {
  it('заменяет кавычки на ёлочки', () => {
    expect(typograf('"тест"')).toBe('«тест»');
  });

  it('ставит неразрывный пробел перед ₽', () => {
    expect(typograf('2700 ₽')).toContain('\u00a0₽');
  });

  it('заменяет двойной дефис на длинное тире', () => {
    expect(typograf('Текст -- продолжение')).toContain('—');
  });
});
```
